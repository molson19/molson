Spring Boot is an amazing framework that allows you to
bootstrap your Java application incredibly fast. In this article, I'll
show you how to create a Spring Boot application using Spring Initializer,
Maven, and GitHub.

# Spring Initializer

Spring Initializer is a web application created by Spring that generates an entire Spring project strucutre
for you. All of the folder structure, dependency management, and config files are
created from the easy to use web interface.

To get started go to https://start.spring.io/.
